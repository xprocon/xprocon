## About Me

Builder focused on backend systems and web technologies. Based in Nanchang, China and currently associated with Mysteel. Active on GitHub since 2017 with a consistent builder-oriented contribution style centered on direct commits.

Primary work involves Java-based microservices, Spring ecosystem projects, frontend implementations with Vue, and static site development using HTML & CSS. Recent activity shows focused development across a small set of personal and utility repositories.
